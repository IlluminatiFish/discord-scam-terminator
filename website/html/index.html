This application is for server moderators to monitor threats on their server via the Discord Scam Hunter bot. You can see the source code for the bot here: https://github.com/shatindle/discord-scam-terminator/. You can also review the public project kanban board.

We use single sign-on to lookup what servers you moderate, then show you a dashboard of removed phishing attempts since the bot was created on Dec 25, 2021.

If you run into issues, please open a bug!
